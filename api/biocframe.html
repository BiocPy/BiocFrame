class biocframe.BiocFrame.BiocFrame(data: MutableMapping[str, Sequence | MutableMapping] | None = None, number_of_rows: int | None = None, row_names: Sequence[str] | None = None, column_names: Sequence[str] | None = None, metadata: MutableMapping | None = None)[source]#

Bases: object

BiocFrame is an alternative to DataFrame, with support for nested column types.

Columns may extend Sequence, and must implement the length (__len__) and slice (__getitem__) dunder methods. This allows BiocFrame to accept nested BiocFrame objects as columns.

Typical usage example:

To create a BiocFrame object, simply pass in the column representation as a dictionary.

# made up ensembl ids.
obj = {
    "ensembl": ["ENS00001", "ENS00002", "ENS00002"],
    "symbol": ["MAP1A", "BIN1", "ESR1"],
}
bframe = BiocFrame(obj)

Alternatively, you can also specify a BiocFrame class as a column

# made up chromosome locations and ensembl ids.
obj = {
    "ensembl": ["ENS00001", "ENS00002", "ENS00002"],
    "symbol": ["MAP1A", "BIN1", "ESR1"],
    "ranges": BiocFrame({
        "chr": ["chr1", "chr2", "chr3"]
        "start": [1000, 1100, 5000],
        "end": [1100, 4000, 5500]
    ),
}
bframe = BiocFrame(obj)

or slice the object

sliced_bframe = bframe[1:2, [True, False, False]]

Subset the data frame.

This operation returns a new object with the same type as caller. If you need to access specific rows or columns, use the row() or column() methods.

Usage:

# made up chromosome locations and ensembl ids.
obj = {
    "ensembl": ["ENS00001", "ENS00002", "ENS00002"],
    "symbol": ["MAP1A", "BIN1", "ESR1"],
    "ranges": BiocFrame({
        "chr": ["chr1", "chr2", "chr3"]
        "start": [1000, 1100, 5000],
        "end": [1100, 4000, 5500]
    ),
}
bframe = BiocFrame(obj)

# different ways to slice.

biocframe[0:2, 0:2]
biocframe[[0,2], [True, False, False]]
biocframe[<List of column names>]
Parameters:

args (SlicerArgTypes) –

A Tuple of slicer arguments to subset rows and columns. An element in args may be,

  • List of booleans, True to keep the row/column, False to remove.

    The length of the boolean vector must be the same as number of rows/columns.

  • List of integer positions along rows/columns to keep.

  • A slice object specifying the list of indices to keep.

  • A list of index names to keep. For rows, the object must contain unique

    row_names and for columns must contain unique column_names.

  • An integer to subset either a single row or column index.

    Alternatively, you might want to use row() or column() methods.

  • A string to subset either a single row or column by label.

    Alternatively, you might want to use row() or column() methods.

Raises:

  • ValueError – Too many slices provided.

  • TypeError – If provided args are not an expected type.

Returns:

  • If a single row is sliced, returns a dict.

  • If a single column is sliced, returns a list.

  • For all other scenarios, returns the same type as caller with the subsetted rows and columns.

Return type:

Union[“BiocFrame”, dict, list]
